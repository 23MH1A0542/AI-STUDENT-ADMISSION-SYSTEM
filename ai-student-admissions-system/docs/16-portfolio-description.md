# 16. Portfolio Description

## Portfolio Project Description
**AI-Enabled Student Admissions & Support Management System**
A fully declarative Salesforce application (no Apex) that centralizes student admissions for training
institutes — covering the full lifecycle from inquiry to enrollment, discount approval, and post-enrollment
support — augmented with an Agentforce AI agent that summarizes applications, flags risk, and recommends
discount decisions under strict, human-in-the-loop guardrails. Includes a two-tier security model, 4
automated Flows, an Approval Process, and 5 role-specific dashboards.

## Project Summary (1 paragraph)
This project demonstrates a production-style Salesforce implementation for student admissions management,
combining a clean declarative data model, layered security, multi-branch automation (Flows + Approval
Process), and Agentforce AI — all built without custom code, showing how far Salesforce's native
declarative tools can go for real enterprise workflows.

## One-Minute Elevator Pitch
"I built a Salesforce system that helps training institutes manage the entire student admissions process —
from first inquiry to enrollment — without a single custom code file. It automatically routes high-discount
enrollments to managers for approval, sends reminder emails for anything sitting too long, and uses
Agentforce, Salesforce's AI agent platform, to summarize applications and flag risk for counselors — while
making sure the AI can only *recommend*, never *approve*, so a human is always in control. It's a good
example of building enterprise-grade automation purely with Salesforce's declarative toolkit."
